# PixelMatter project plan

## Done

- C++20/raylib desktop sandbox.
- Shared UI geometry for drawing and hit-testing.
- Fixed material panel hover/click drift.
- Resizable window and F11 fullscreen toggle.
- Material-aware brush fill and continuous drawing.
- Materials: Sand, Water, Mud, Stone, Wood, Fire, Smoke, Steam, Lava, Acid, Metal, Glass, Oil, Plant, Ice, Electricity.
- Improved Mud/Ice/Electricity interactions.
- Stable save material IDs with versioned load mapping.
- CTest smoke/unit tests.
- Clean release archive rules.

- Temperature view mode.
- Pressure/flow map for liquids.
- More materials: salt, gunpowder, concrete, snow.
- Web version using the simulation rules as reference.
