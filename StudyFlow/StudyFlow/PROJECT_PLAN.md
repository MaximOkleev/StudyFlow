# StudyFlow project plan

## Done

- Kotlin Compose Desktop application.
- Dashboard, subjects, tasks, calendar, notes, timer, statistics and settings.
- Editable calendar tasks.
- Deadline input via ISO date.
- Filters by status, subject and priority.
- Real-time focus timer with beep and session logging.
- Atomic local persistence and validated raw restore.
- Timestamped exports.
- Responsive compact sidebar and saved window size.
- Basic JVM unit tests.
- README with run/test/package instructions.

## Next

- Replace Properties persistence with SQLite/SQLDelight.
- Add real DatePicker UI.
- Add drag-and-drop Kanban view.
- Add recurring tasks and daily habits.
- Add richer desktop notifications.
- Add more Compose UI tests.
